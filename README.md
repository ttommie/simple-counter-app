# Simple Counter App

A simple counter project created using react.

## Features

-   Real-time Input Updates.
-   Live Preview
-   Mobile Accessible

## Deployment

Clone or Download

```bash
git clone https://github.com/ttommie/simple-counter-app.git
```

Install all node modules

```bash
npm install
```

Run the react server

```bash
npm start
```
